User Guide

  • This is a very basic calculator with limited functionalities.
  • You can perform + , - , *, /
  • To get the square root of any number: enter √number , √25 = 5
    Other statements might give an error
  • To get the % of any numbe Use: number %of number
    Ex: 25 %of 300 = 75
  • Use "C" to clear the display.
